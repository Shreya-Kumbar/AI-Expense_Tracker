# 💵 AI Expense Tracker

AI Expense Tracker is a full-stack web application that helps users track and manage their daily expenses easily.

Users can:
- Add expense title
- Select expense category
- Enter expense amount
- View total expenses instantly
- Store expense data permanently in a database

## » Features

- Add and delete expenses
- Category-based expense tracking
- Automatic total expense calculation
- Clean and responsive UI
- Data stored using MySQL database

## » Tech Stack

### Frontend
- React.js

### Backend
- Node.js
- Express.js

### Database
- MySQL

## » How It Works

1. User enters:
   - Expense title
   - Amount
   - Category

2. Expense is sent to the backend using API requests.

3. Backend stores the data in MySQL database.

4. Frontend fetches and displays all expenses along with the total amount.
